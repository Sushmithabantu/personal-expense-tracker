# Personal Expense Tracker

A console-based Personal Expense Tracker built using Python.

## Features
- Add income and expense transactions
- Store data using CSV file handling
- View complete transaction history
- Automatic summary calculation (Total Income, Total Expense, Balance)
- Clear all stored data

## Technologies Used
- Python
- CSV Module
- Datetime Module

## How to Run
1. Clone the repository
2. Run: python expense_tracker.py
